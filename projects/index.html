Document Intelligence Web Application
(Azure Cognitive Service)

With Microsoft Azure Cognitive services (Cognitive search / Form recognizer ) , I am able to build an intelligent document management solution that takes documents of all formats and size and extract needed information from them. This is a python Flask web-application that calls serveral azure cognitive services through REST-APIs. The search service rides on Azure cognitive search which goes through a repository (Azure Blob Storage) and extract text information from every document contained whithin. The document intelligence rides on trained models from azure form recognizer where I train models with a few documents to be able to extract certain content automatically.

Brand Reputation Management System (Web Platform)

This is a social media analysis tool, that sraps data from all social media channels (Facebook, Twitter, Instagram, LinkedIn, and Blogs) about a brand an measures the society's perception of the brand by analysing sentiments of posts sent. I built this application with Python FLASK. It is equipped with some web scrapping and database interacting agents. The data is scrapped from the web, and stored in a database then the application retrieves the data and displays needed information to the user
